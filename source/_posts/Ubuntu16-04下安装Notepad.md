### 安装方法

```bash
sudo add-apt-repository ppa:notepadqq-team/notepadqq

sudo apt-get update

sudo apt-get install notepadqq
```

### 卸载方法

```bash
sudo apt-get remove notepadqq

sudo add-apt-repository --removeppa:notepadqq-team/notepadqq
```
